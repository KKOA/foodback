## Set up Application
This installation assumes have 
- MySQL
- PHP7
- Composer

installed.

```
git clone https://github.com/KKOA/foodback.git
cd foodback
composer update
php artisan key:generate
```

## Run Application
```
cd foodback
php artisan serve
```
Click the link in command prompt/terminal to open application in the browser.
